## 핵심 KPI (전/후 비교)

| 지표 | 기준 | 적용 전 | 적용 후 | 판정 |
|------|------|---------|---------|------|
| IMAGE 평균 duration | ≤ 기준 | - ms | - ms | ⬜ OK / ⬜ WARN |
| IMAGE p95 duration | 안정/↓ | - ms | - ms | ⬜ OK / ⬜ WARN |
| IMAGE 실패율 | +2%p 이내 | - % | - % | ⬜ OK / ⬜ STOP |
| 첫 요청 통과율 | ≥ 99% | - % | - % | ⬜ OK / ⬜ STOP |
| 체감 제한 비율 | ≤ 3% | - % | - % | ⬜ OK / ⬜ WARN |

**판정 기준:**
- **OK**: 기준 충족
- **WARN**: 기준 근접 (주의 필요)
- **STOP**: 기준 미충족 (적용 금지)
